# react-native-mrz-reader

Reads MRZ field for React Native

## Installation

```sh
npm install react-native-mrz-reader
```

## Usage

```js
import { MrzReaderView } from "react-native-mrz-reader";

// ...

<MrzReaderView color="tomato" />
```
